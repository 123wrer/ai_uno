# AI Uno

Этот проект обучает нейросеть (ИИ) играть в Уно, используя Reinforcement Learning (Обучение с подкреплением).

## Как запустить на своем компьютере

1. Установите необходимые библиотеки (если еще не сделали):
   ```bash
   pip install rlcard[torch] matplotlib
   ```

2. Запустите скрипт обучения:
   ```bash
   python train.py
   ```

## Как ИИ учится, не зная действий второго игрока?

В обучении с подкреплением (Reinforcement Learning) наш ИИ играет против **Среды** (Environment). 
* Среда берет на себя все правила игры: она тасует колоду, проверяет легальность ходов и **она же делает ходы за противника**.
* Наш ИИ видит **только свои карты и карту на столе**. Он не знает, какие карты у противника и не видит его логики.
* Это называется **игрой с неполной информацией**. ИИ делает свой ход, отдает его Среде, а Среда возвращает новый результат стола (после того как сходил противник). ИИ учится исключительно на результатах (победа или поражение).

## Поддержка видеокарт (Vulkan / RX 590)

В коде есть проверка на Vulkan.
* PyTorch в основном обучает модели через CUDA (Nvidia) или ROCm (AMD Linux). Поддержка Vulkan у них экспериментальная и в основном предназначена для телефонов.
* Скрипт попытается запустить Vulkan. Если ваша сборка PyTorch его не поддерживает для тензоров обучения, он **автоматически переключится на ваш процессор (CPU)**.
* Для такой игры как Уно нейросеть очень маленькая, и ваш процессор обучит её за несколько минут! Мощность видеокарты здесь даже не обязательна.
